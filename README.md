# testinium
Web Automation Project

"www.gittigidiyor.com" sitesi üzerinden senaryo testlerinin yapıldığı Selenium Web Otomasyon projesidir.

Site üzerinde gerçekleştirilen testin adımları;

1. Ana sayfanın açıldığı kontrol edilir ve siteye login olunur.
2. Login işlemi kontrol edilir.
3. Arama kutucuğuna "bilgisayar" kelimesi girilir.
4. Arama sonuçları sayfasından 2. sayfa açılır.
5. Sayfa 2'nin açıldığı kontrol edilir.
6. Sonuca göre sergilenen ürünlerden rastgele bir ürün seçilir.
7. Seçilen ürün sepete eklenir.
8. Ürün sayfasındaki fiyat ile sepette yer alan ürün fiyatının doğruluğu karşılaştırılır.
9. Adet arttırılarak ürün adedinin 2 olduğu doğrulanır.
10. Ürün sepetten silinerek sepetin boş olduğu kontrol edilir.
